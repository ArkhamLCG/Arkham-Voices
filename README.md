## Arkham Voices

**Arkham Voices** is a website where you can **listen to and download narration** for *Arkham Horror: The Card Game* campaigns and scenarios.

### Website

- `https://arkhamlcg.github.io/Arkham-Voices/`

### How to use

- **Open the website** and choose your **language** in the top bar.
- Pick a **campaign** and then a **scenario**.
- Use the **player** to listen, and download audio when available.

### What’s included

- **Fan-made campaigns**: available directly on this site.
- **Official campaigns**: narration is provided by the **Dissonant Voices** project.
  - To use it, you’ll need a **Patreon subscription**: `https://www.patreon.com/MythosBusters`
  - Playback is available in the **ArkhamCards** app and on the Dissonant Voices website:
    - `https://arkhamcards.app/`
    - `https://dissonantvoices.north101.co.uk/`

### Tech stack

- **Vite** + **React** + **TypeScript**
- **MUI (Material UI)** for UI components/theme
- **Wouter** for routing
- **i18next / react-i18next** for translations
- **Biome** for linting/formatting
- **GitHub Pages** deployment via GitHub Actions
